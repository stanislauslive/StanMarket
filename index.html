StanMarket by stanislauslive

Development For Apps

Introduction of the oscMall System as a complete management tool for webmasters who want to host multiple shops on their site.

Dependencies

OsCommerce ALOT of ajax changes to the admin

Install

A list of everything that needs to be set with 755/777 permissions for the operations of the oscmall system. Files includes/configure.php mall_admin/includes/configure.php includes/languages/english/mainpage.php folders images/ images/1_images includes/languages/english if enabling banners set images/banners 1_images/banners mall_admin/imgages/graphs and all files inside the /graphs folder For backups mall_admin/backups
